>  本项目为[pk5ls20-COS_uploader](https://github.com/pk5ls20/COS_uploader)的GUI重构版，使用Pyqt5重新开发
### 已实现/新增的功能
- [x] 加载/保存参数文件
- [x] 上传单个文件/文件夹
- [x] 上传进度条
- [x] 状态栏指示
- [x] 日志分级保存
- [x] 更多在GUI里特有的小细节...
### 已知问题
- [x] 拖拽系统路径（如回收站）程序崩溃
- [x] 上传含大量文件的文件夹时崩溃
